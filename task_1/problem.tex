Минимизировать функции
\begin{itemize}
    \item $f_1(x) = 100(x_2 - x_1^2)^2 + 5(1 - x_1)^2$;
    \item $f_2(x) = (x_1^2 + x_2 - 11)^2 + (x_1 + x_2^2 - 7)^2$;
    \item где $x = (x_1, x_2)\in\mathbb{R}^2$.
\end{itemize}

используя следующие методы:
\begin{enumerate}
    \item Метод дефорсмируемого многогранника;
    \item Метод градиентного спуска;
    \item Метод сопряженных градиентов;
    \item Метод Ньютона;
\end{enumerate}
